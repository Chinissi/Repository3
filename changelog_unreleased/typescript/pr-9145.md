#### Do not line break union types where all but one option are keywords and contain `undefined` (#9145 by @ehrencrona)

<!-- prettier-ignore -->
```typescript
// Input
type T = {
  foo: string,
} | undefined;

// Prettier stable
type T =
  | {
      foo: string;
    }
  | undefined;

// Prettier master
type T = {
  foo: string;
} | undefined;
```
