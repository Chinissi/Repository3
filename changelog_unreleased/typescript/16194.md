#### Yield expression parens (#16194 by @kirkwaiblinger)

Add parens around yield expressions if parent is an angle-bracket type assertion.

<!-- prettier-ignore -->
```ts
// Input
function* g() {
  const y = <T>(yield x);
}

// Prettier stable
function* g() {
  const y = <T>yield x;
}

// Prettier main
function* g() {
  const y = <T>(yield x);
}
```
