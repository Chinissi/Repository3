#### Format HTML attribute `style` value ([#7556](https://github.com/prettier/prettier/pull/7556) by [@fisker](https://github.com/fisker))

<!-- prettier-ignore -->
```html
<!-- Input -->
<div style="   color : red;
            display    :inline ">
  </div>

<!-- Prettier stable -->
<div
  style="   color : red;
            display    :inline "
></div>

<!-- Prettier master -->
<div
  style="
    color: red;
    display: inline;
  "
></div>
```
