#### Respect single-quote option in HTML attributes (#11559 by @dcyriller)

<!-- prettier-ignore -->
```sh
--single-quote
```

<!-- prettier-ignore -->
```html
<!-- Input -->
<div title='My title'></div>
<div title="My title"></div>

<!-- Prettier stable -->
<div title="My title"></div>
<div title="My title"></div>

<!-- Prettier main -->
<div title='My title'></div>
<div title='My title'></div>
```
