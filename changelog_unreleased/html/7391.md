#### Print HTML5 `doctype` in lowercase (#7391 by @fisker)

<!-- prettier-ignore -->
```html
<!-- Input -->
<!DocType html>
<html><head></head><body></body></html>

<!-- Prettier stable -->
<!DOCTYPE html>
<html>
  <head></head>
  <body></body>
</html>

<!-- Prettier master -->
<!doctype html>
<html>
  <head></head>
  <body></body>
</html>
```
