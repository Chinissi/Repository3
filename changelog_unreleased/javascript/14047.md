#### Fix displacing of comments in default switch case (#14047 by @thorn0)

It was a regression in Prettier 2.6.0.

<!-- prettier-ignore -->
```jsx
// Input
switch (state) {
  default:
    result = state; // no change
    break;
}

// Prettier stable
switch (state) {
  default: // no change
    result = state;
    break;
}

// Prettier main
switch (state) {
  default:
    result = state; // no change
    break;
}
```
