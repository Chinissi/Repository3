#### Disable wrapping JSX fragments in parens (#12280 by @user)

Removing parentheses around JSX `<>` fragments reduces the amount of indentation required for elements inside of them.

<!-- prettier-ignore -->
```jsx
// Input
return (
  <>
    <div className="Content">
      <p>Hello, world!</p>
    </div>
  </>
);

// Prettier stable
return (
  <>
    <div className="Content">
      <p>Hello, world!</p>
    </div>
  </>
);

// Prettier main
return <>
  <div className="Content">
    <p>Hello, world!</p>
  </div>
</>;
```
