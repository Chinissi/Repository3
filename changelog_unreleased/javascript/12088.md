#### Inline `await` expressions in JSX (#12088 by @j-f1)

`await` expressions in JSX are now inlined if their argument would be inlined.

<!-- prettier-ignore -->
```jsx
// Input
{await Promise.all(
  someVeryLongExpression
)}

// Prettier stable
{
  await Promise.all(
    someVeryLongExpression
  )
}

// Prettier main
{await Promise.all(
  someVeryLongExpression
)}
```
