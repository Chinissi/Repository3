#### Support formatting for embedded `gql` function call (#14586 by @sosukesuzuki)

<!-- prettier-ignore -->
```jsx
// Input
gql(`
                                 type Foo {
   bar: String!
}
`)

// Prettier stable
gql(`
                                 type Foo {
   bar: String!
}
`)

// Prettier main
gql(`
  type Foo {
    bar: String!
  }
`);
```
