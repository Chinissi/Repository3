#### Fix incorrect handling of comments before else statement ([#8120](https://github.com/prettier/prettier/pull/8120) by [@thecodrr](https://github.com/thecodrr))

```js
// Input
if (cond) {}
/* comment */
else {
  return;
}

// Prettier stable
if (cond) {
} else {
  /* comment */
  return;
}

// Prettier master
if (cond) {
}
/* comment */
else {
  return;
}
```
