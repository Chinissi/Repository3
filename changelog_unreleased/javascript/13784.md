#### Add gql() call expression support (#13784 by @Chnapy)

<!-- prettier-ignore -->
```jsx
// Input
gql(`
      {
    user(   id :   5  )  {
      firstName
      lastName
    }
  }
`);

// Prettier stable
gql(`
      {
    user(   id :   5  )  {
      firstName
      lastName
    }
  }
`);

// Prettier main
gql(`
  {
    user(id: 5) {
      firstName
      lastName
    }
  }
`);
```
