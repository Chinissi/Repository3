#### Do not add trailing comma for grouped scss comments (#15217 by @auvred)

<!-- prettier-ignore -->
```scss
/* Input */
$foo: (
	'property': (),
	// comment 1
	// comment 2
)

/* Prettier stable */
$foo: (
  "property": (),
  // comment 1
  // comment 2,
);

/* Prettier main */
$foo: (
  "property": (),
  // comment 1
  // comment 2
);
```
