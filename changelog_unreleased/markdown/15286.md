#### Do not print trailing commas in arrow function type parameter lists in `ts` code blocks (#15286 by @sosukesuzuki)

<!-- prettier-ignore -->
````md
<!-- Input -->
```ts
const foo = <T>() => {}
```

<!-- Prettier stable -->
```ts
const foo = <T,>() => {}
```

<!-- Prettier main -->
```ts
const foo = <T>() => {}
```
````
