#### Encoding `<` and `>` in markdown urls (#15400 by @vivekjoshi556)

<!-- prettier-ignore -->
```md
<!-- Input -->
[link](https://www.google.fr/()foo->bar)

<!-- Prettier stable -->
[link](<https://www.google.fr/()foo->bar>)

<!-- Prettier main -->
[link](<https://www.google.fr/()foo-%3Ebar>)
```

<!-- prettier-ignore -->
```md
<!-- Input -->
![link](<https://www.google.fr/()foo->bar>)

<!-- Prettier stable -->
![link](<https://www.google.fr/()foo->bar>)

<!-- Prettier main -->
![link](<https://www.google.fr/()foo-%3Ebar>)
```
